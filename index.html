Qué es el W3C

El W3C (World Wide Web Consortium) es una organización internacional que desarrolla estándares para garantizar el crecimiento y la interoperabilidad de la web.

Estándar vs Recomendación

Un estándar es una norma oficial adoptada por el W3C; una recomendación es una propuesta técnica en revisión antes de convertirse en estándar.

Validación con W3C

Puedes validar tu sitio web en W3C Validator para confirmar que cumple con las reglas de HTML y CSS. Si ves el mensaje "Document checking completed. No errors found.", tu código está correcto.
